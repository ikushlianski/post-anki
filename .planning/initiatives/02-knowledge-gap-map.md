# [Initiative] Knowledge Gap Map

**Priority:** medium
**Phase:** 2 (only after Initiative 1 proven for 4+ weeks)

## What this unlocks for the user

A visual map of what the user doesn't know, tied to their actual stack. Instead of
telling you what to learn, the system shows you the shape of your ignorance:
"You work with Lambda daily but have zero recorded gaps about cold starts."

The user can steer their growth deliberately, dismiss things that don't matter to
them, and see patterns across tools (security is consistently weak across 3 tools).

## Phase gate

This initiative MUST NOT start until:
- Phase 1 (Daily Architect Mentor) has been actively used for 4+ weeks
- At least 20 gaps have been logged from real discussions
- User has confirmed Phase 1 experience is valuable

Building a dashboard before these conditions are met produces an empty dashboard
nobody uses.

## Depends on

[Initiative] Daily Architect Mentor — gap data must exist before a map is useful

## Enables

[Initiative] Judgment Training — /decide mode benefits from gap map context

1. [Epic] Web dashboard for gap triage
