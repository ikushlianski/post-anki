# [Initiative] Daily Architect Mentor

**Priority:** urgent
**Phase:** 1 (MVP — prove the loop before building anything else)

## What this unlocks for the user

A senior architect mentor that shows up in Telegram every day, asks the user a
question they have to think about, and quietly tracks what they don't know.
No card queue. No guilt. No management overhead.

The user experiences: daily intellectual challenge, gradual awareness of their own
knowledge gaps, and growing ability to reason through architectural tradeoffs —
without ever feeling like they're doing "Anki homework."

## Anti-Anki contract (enforced at this initiative level)

- User never manages a queue of any kind
- System always initiates — user never has to remember to study
- No response is met with complete silence, not a reminder
- No trivia questions — every question requires reasoning, not recall
- Maturity/scores are invisible in Phase 1 — they exist in the database but are not shown

## Success signal

User engages with at least 4 daily pushes per week after 2 weeks of use — WITHOUT
the system reminding them or tracking streaks. Engagement happens because the
questions are genuinely interesting, not because of guilt.

## Depends on

None — this is the foundation initiative.

## Enables

- [Initiative] Knowledge Gap Map (only start after this is proven for 4+ weeks)
- [Initiative] Judgment Training (only start after gap tracking has data)
- [Initiative] Ecosystem Awareness (only start after user is actively engaged)

1. [Epic] System-initiated Socratic loop via Telegram
