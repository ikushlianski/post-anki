# Rejected Alternatives

## [2026-04-19] "Immediate" as a third cadence mode
**Rejected:** A distinct "immediate" trigger separate from on-demand.
**In favor of:** Decision: Cadence: daily push + on-demand only.
**Why:** On-demand already covers it; a third mode adds complexity with zero new behavior.

## [2026-04-19] AI-created gaps (PRD original spec)
**Rejected:** Gaps auto-logged by AI when a Socratic answer is wrong or shallow.
**In favor of:** Decision: Gap creation is user-only.
**Why:** Auto-logging silently recreates Anki accumulation and guilt — the core failure mode this product exists to avoid.
